NID Application System

Bangladeshi citizens with an existing NID can apply for Correction/Reissue. Citizens who do not have an NID, can apply for a new NID. Those who have already applied but have not received the NID card yet, can register the account and get a copy of the NID card from the download option.

NID Wallet

NID Wallet is a multi-purpose application developed under Bangladesh Election Commission. The goal of this application is to make sure the NID related services are easily available on a phone.

DataHub

DataHub is a project operated by the National Telecommunication Monitoring Center (NTMC). It aims to streamline citizen information monitoring by centralizing data from various sources into a single hub. The project includes monitoring and managing diverse information, such as passport records, National ID (NID) details, birth registrations, Mobile Operator data, criminal records, and driving licenses. This project also includes storing and identifying Rohingya refuses
